Siga este passo a passo para obter sua chave de API, necessária para a integração da SDK no seu aplicativo.

## 1. Ajustes do aplicativo

1. Faça login na [plataforma](https://app.advents.io) da Advents.
2. Selecione o aplicativo desejado.
3. Navegue para a seção `Ajustes` do seu aplicativo.

![Ajustes](/assets/images/settings.png)

## 2. Chave da API

Dentro das configurações do seu aplicativo, acesse a seção `SDK`.

![Chave da API](/assets/images/api-key.png)

Utilize a chave de API para inicializar a SDK instalada no seu aplicativo, conforme instruções do [guia de integração](/sdks).

<Warning>Mantenha sua chave de API segura e não a compartilhe publicamente.</Warning>
